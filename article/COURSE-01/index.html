视频播放问题解决指南

准备工作

  1. 一个会识字的人
  2. 一台电脑
  3. 一个浏览器（推荐使用必应搜索 官网）

附加工具（可能用不到）

  1. 下载工具（推荐 IDM 破解版链接）
  2. GitHub 下载加速（这是我的镜像网站 镜像）
  3. 广告拦截插件（AdGuard Edge 浏览器下载链接）

视频播放时可能出现的提示

当我们打开视频文件时，可能会出现解码错误。
提示内容：播放此视频需要新的编解码器，价格 7 元。
这是因为我们打开的视频是以 H.265（又名 HEVC）编码的。那么，什么是视频编码呢？

视频编码简介

视频编码方式是指通过特定的压缩技术，将某个视频格式的文件转换成另一种视频格式文件的方式。有关当前主流视频编码技术的详细分析，请参见以下链接：

  • 当前主流视频编码技术浅析
  • 下一代视频编码标准 H266

解码方式

播放视频时需要解码，解码分为两种：

  1. 软件解码：由 CPU 处理，适合多种任务，但速度较慢。
  2. 硬件解码：由专用电路芯片处理，速度快但功能单一。
    有关软件解码与硬件解码的更多信息，请参考 知乎。

解决方案

有两种方法（任选其一）可以解决视频播放问题：

方法一：安装播放器（软件解码）

推荐安装以下播放器：

  • KMPlayer
  • PotPlayer
  • VLC
  • MPV

如何选择正确的版本

  • Alpha 版：主要用于软件开发者内部交流，存在较多 Bug。
  • Beta 版：相对于 Alpha 版有较大改进，但仍有缺陷，主要修改对象是软件的 UI。
  • RC 版：相当成熟，基本不存在导致错误的 Bug。
  • Release 版：最终版本，交付用户使用的版本。

如何选择适合你系统的版本

  • Windows：选择带有 windows 标识的安装包，如 KMPlayer-x86-win.exe。
  • macOS：选择带有 macOS 标识的安装包，如 KMPlayer-x64-mac.dmg。
  • Linux：选择带有 Linux 标识的安装包，如 KMPlayer-amd64-linux.tar.gz。
    其中：
  • x86 表示 32 位系统
  • x64 表示 64 位系统
  • amd64 表示 64 位系统（AMD 首先推出 64 位架构）
    如果下载的是压缩包，请解压后运行安装包。

如何下载压缩软件？

下载压缩软件时，常见的压缩包后缀名有不同，详细信息请见：

  • 了解数据压缩算法
  • 十款性能最佳的压缩算法
    结论：下载并安装 7-Zip。

方法二：硬件解码

  1. 浏览器中输入 https://store.rg-adguard.net/
    • 将左边的选项改为 ProductId
    • 输入 9n4wgh0z6vhq
    • 点击 √
      根据电脑型号选择版本，然后下载安装。
  2. 访问 https://www.free-codecs.com/hevc_video_extension_download.htm，选择所需版本下载安装。

其他方法

如果以上方法都无法解决问题，可以使用网上的解析站点：

  1. 安装篡改猴 Tampermonkey：油猴是一款用户脚本管理器，可以通过编写的脚本来增强浏览器功能。
    • 进入油猴官网 官网
    • 点击下载，页面会跳转至 Edge 外接扩展商店的安装页面，直接获取安装即可。
    • 对于其他浏览器，可以通过搜索 chrome://extensions/ 进入扩展程序页面，开启开发者模式，拖入解压后的 tampermonkey.crx 文件完成安装。
  2. 获取新脚本：安装成功后，点击右上角的油猴插件 logo，进入脚本市场。
    • 推荐使用 GreasyFork ，搜索 VIP，安装下载人次较高的脚本。
  3. 播放视频：打开爱奇艺、腾讯视频等视频网站，观看视频时，会出现一个按钮，选择合适的解析源播放即可。
